# Teamified-RAG-Pipeline
A Retrieval-Augmented Generation(RAG) pipeline that answers questions about Philippine history by semantically searching a PDF document and feeding relevant context to an OpenAI language model.

# Setup & Run Instructions

commands in VS code
Step-0: Clone the repo and navigate the folder using cmd/VS code
Step-1: Create a Virtual Environment(venv) (Python 3.11)
Step-2: Run .\venv\Scripts\Activate.ps1
Step-3: Run pip install -r requirements.txt
Step-4: Run python.exe run.py

# LLM and embedding choices

model:'gpt-40-mini'
Embedding Model: "text-embedding-3-small"
